DoctrineFormGenerator

Provides the ability to dynamically generate Forms from Doctrine2 models in Symfony2

View project onGitHub

Objective

When you create a Doctrine2 model, you define your database schema. In order to utilize your new model in a CRUD action, you need to create a Form, and redefine each field, and how it should be validated. DoctrineFormGenerator aims to prevent this duplication of work by allowing you to dynamically generate your form from your model, preventing the need to create additional form definitions for your CRUD operations.

Implementation

Initial implementation of the project will provide the ability to dynamically create a form at run-time via the Symfony2's createForm method passed with no parameters. This form will be based off the Doctrine2 model's field schema.

Next steps

Support for other field schema definition methods, YAML XML and PHP definitions.
